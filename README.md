# 🚀 Usage

## Full Autonomous Scan

```bash
./reconx.sh -d target.com --autopilot
```

## Behavioral Analysis

```bash
./reconx.sh -d target.com -m behavioral
```

## Aggressive Recon

```bash
./reconx.sh -d target.com --mode aggressive --parallel
```
